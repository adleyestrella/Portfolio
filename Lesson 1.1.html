Lesson 1.1

Introduction to Hyperlinks

This lesson introduces hyperlinks which are references that lets users go to another page, document, or location on the same page. It is created using the <a></a> anchor tag with the href attribute that contains the link’s destination. The anchor tag has 3 attributes: href. target, and name. Links can appear as unvisited link, visited link, and active link.

Hyperlinks have three states: hover, active, and visited. The three types of hyperlinks are absolute URL, relative URL, and named anchor. The target attribute has 4 values: blanlk,_self, _parent, and _top. Links can open in different ways using the target attribute and can be accessed using a mouse pointer, Tab key, or access key.

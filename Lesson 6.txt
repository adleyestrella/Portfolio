<p> <span>Pseudo classes</span> are not user defined selector but defined by the colon and it is used on specific HTML elements only. In the anchor element, it has 4 pseudo classes: link, hover, visited and active. Each one of them affects the anchor element's behavior.</p>
<div class="container1">
<img src="code12.jpg" class="gif">
<p>The<span>linked pseudo class</span> is for unvisited hyperlinks.<br>
The <span> visited pseudo class </span> is for visited hyperlinks.<br>
The <span> hover pseudo-class </span> is used for hyperlinks that when the mouse hovers over the link, it will style the element based on the CSS code you have written.<br>
The <span>active pseudo-class</span> is used for hyperlinks that when it is clicked, it will style the element based on the CSS code you have written.
</p>

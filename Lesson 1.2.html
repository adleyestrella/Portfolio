Hyperlinks

Hyperlinks use the target attribute to control where the linked page opens. _self opens the link in the same browser window, while _blank opens it in a new window or tab. Hyperlinks can also connect to pages on other websites by using the full URL of the destination page.

A named anchor allows linking to a specific part of the same page using the name attribute and # in the link. Named anchors are invisible to users.

Email links let users send an email directly by using mailto: in the href attribute.

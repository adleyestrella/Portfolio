Lesson 2

More about CSS

In this lesson, it talks about the proper structure of CSS, the stylesheets, and the kinds of CSS. The structure helps us know where do we put the element that we want to style specifically like values. The stylesheets helps the Web browser to show the style of the element based from the CSS codes we put. There are 3 types of CSS: inline, external, and linked. Each types have their own way in distributing the styles to the element and placed in specific places that can be used for putting CSS codes.
